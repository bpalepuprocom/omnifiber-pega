Welcome to Omni Fiber's Order Management documentation!
===================================

**Order Management** is a Pega application which orchestrates the orders of Omni Fiber to cater different channels this application offers a *simple* and *intuitive* REST API.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
